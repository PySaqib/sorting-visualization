How to use this tool?

Firstly, user can fill the shown array with random, unsorted numbers using "FILL ARRAY" button. Next, you can use one of the provided algorithms to implement. You can also adjust the speed of execution using the speed counter given above.

What is sorting?

According to Wikipedia, sorting is a process of arranging items systematically. In computing, sorting is referred to as categorizing, listing, or simply arranging data in a manner that may be meaningful to the computer or not. Sorting can also be done among elements with common attributes.

Sorting Algorithms

To cope with numerous computing challenges, mathematicians and computer scientists have devised various sorting algorithms. Famous ones include Bubble Sort, Selection Sort, Insertion Sort & QuickSort.

In this tool, the user can visualize how these algorithms analyze the given data, draw comparisons, and sort the elements.
